- this project using TYPESCRIPT + SWC, which is a lil different than only TYPESCRIPT, so need to be careful
- every component need testing FIRST !


before start, after install vite
1. npm install -D vite typescript @vitejs/plugin-react
2. run server => npx vite

| Test Type       | Tool                          |
| --------------- | ----------------------------- |
| Unit Tests      | Vitest                     |
| Component Tests | React Testing Library      |
| E2E Tests       | Cypress or Playwright  |
